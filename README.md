# Library-Manager-App

## How to Run 
- Clone Project in IntelliJ IDEA
- Start Apache and MySQL from XAMPP
- Run the project and visit localhost:8080

## Functionalities
### Authors
- View Authors
- Add new Author
- Delete/Update Author
- View All Authors with Their Books

### Books
- View Books
- Add new Book
- Delete exixting book
- Update Book Data
